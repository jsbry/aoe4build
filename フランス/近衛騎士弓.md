# 近衛騎士弓


## オーダー

- 初期農民は全員羊
- 食糧8まで羊へ
- 生産農民4人を金へ
- 進化押しまで生産農民は羊へ
- 4人で騎兵学校進化。食糧農民を1人はがし、果実横に粉ひき所を建てる
- 手押し車を研究する
- 生産農民2人を伐採へ
- 食糧9/伐採2/金4へ
- 進化完了。進化農民4人を食糧に戻す
- 食糧13/伐採2/金5へ
- 近衛騎士を連続生産していく
    - 食糧13/金5が連続生産の目途
- 早めに内政テクを入れていく
    - 金10くらいまで増やしつつ？
- 敵の槍が少なそうなら騎兵小屋2つ目、多そうなら弓小屋を建てる
    - 弓回すなら金5まで減らして伐採に回すなど
- 頃合いを見て騎士道の研究を入れる
    - 2~3体瀕死の騎士出たら
    - ラッシュバトルのときは後回し
